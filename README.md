# Login Page UI

A clean and modern Login Page UI created as part of Task 3 for the Synent Technologies Web Development Internship Program.

## 📌 Task Details

**Task 3: Login Page UI**

### Objective

Design a clean and properly aligned login interface.

### Requirements

* Email and Password fields
* Login button
* "Forgot Password?" text

### Output

* Modern and properly aligned UI

---

## 🚀 Features

* Responsive and modern login form
* Email validation (`@` required)
* Password show/hide toggle
* Login authentication using predefined credentials
* Success and error messages
* Forgot Password functionality

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript

4. Click the Login button.

---

## 📸 Functionalities

### ✅ Login Validation

* Checks if all fields are filled.
* Ensures the email contains `@`.
* Verifies email and password.

### 👁 Show/Hide Password

* Click the eye icon to show or hide the password.

### 🔐 Forgot Password

* Prompts the user to enter a registered email.
* Simulates sending a password reset link.
